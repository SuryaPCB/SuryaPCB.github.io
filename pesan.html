Ukuran Default Desain

Ukuran default yang kami gunakan adalah sebagai berikut : 
Layer : 1 layer

Ukuran jalur : 32 mil (10 mil untuk komponen SMD)
Jarak Antar Jalur : 20 mil (untuk komponen SMD menyesuaikan)
Ukuran resistor : 1 layer dan 2 layer DIP menggunakan ukuran 1207 sedangkan untuk SMD menggunakan ukuran 0805
Ukuran komponen lainnya menyesuaikan.
